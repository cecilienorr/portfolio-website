pete's pizzaria

Pete's Pizzaria was a game we had to create in the end of 1st semester. The goal was to learn CSS animation, basic JavaScript and illustrations with certain requirement.

I created an activity diagram to structure the way I later on coded the game. Alongsidestyletile making a styletile with all my buttons and screens that would pop up when you either won or lost the game.

In this project we had some design requirements. We had to illustrate everything on our own and we had to illustrate figures such as triangles, squares, circles, etc. Outlines were forbidden in this project.
